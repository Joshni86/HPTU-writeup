# $1.5 Billion

**Points**: 495  
**Solves**: 1  
**Flag Format**: HPTU{0x____D}

## Solution

1. Identify a famous $1.5B crypto event — e.g., **Bitfinex 2016 hack** (~120k BTC).
2. Trace transaction history or smart contract related to that incident.
3. Locate an Ethereum address ending in `D` buried in contract storage or transaction input data.
4. Use Etherscan to interact with the contract — call `owner()` or read public variables until you find an address ending in `D`.
5. **Flag example:** `HPTU{0x742C2FD6D9B1B6D3D3e5D4a4eD7a8b9c0d1e2f3D}`
